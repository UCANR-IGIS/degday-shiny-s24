Fact or Fiction?

All weather data is created equal

Hourly data is better than daily data.

Closer is better.

If the station is far away, an interpolated value at the farm is better than the measured value at the station.

Free weather data is better.

OpenMeteo

Access

No API key is required if:

  • for non-commercial use, and
  • less than 10.000 daily API calls


APIs

Forecast API

Hourly and daily data available

By default, we provide forecasts for 7 days, but you can access forecasts for up to 16 days.

If you’re interested in past weather data, you can use the Past Days feature to access archived forecasts.

For USA, combines the NOAA GFS weather model with the rapid updating HRRR weather model

Historic Weather API

You can access past weather data dating back to 1940. However, there is a 5-day delay in the data. If you want information for the most recent days, you can use the forecast API and adjust the Past Days setting.
